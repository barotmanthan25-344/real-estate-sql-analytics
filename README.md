# 🏠 Real Estate Market Analytics (SQL Server)

## Overview

An advanced SQL analytics project built on 16,300+ real estate records across residential listings, rental markets, and transaction trends.

The project demonstrates advanced SQL techniques including:

- Window Functions
- Common Table Expressions (CTEs)
- Recursive CTEs
- PIVOT
- Statistical Analysis
- Business Intelligence Reporting
